## 简介
一个基于Vue3的个人博客。 喜欢给个Star。 支持响应式，不过目前只对手机端进行了优化。支持一件切换主题，目前只有白天黑夜两种。

源码地址：https://gitee.com/jjhroy/mongo-blog

## 更新内容

用Vue3将之前的博客进行了重写,比起Vue2版本删去很多实验性的功能，云盘、爬虫、相册、支付模块等,优化一些逻辑。

后台将会用React重写，不过上线时间可能要等到年后了。
